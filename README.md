# Website Simple

## About

This project is a simple template website created using HTML and CSS. It serves as a starting point for developing responsive and visually appealing web pages.

## Features

- Clean and modern design
- Responsive layout
- Easy to customize
- Basic navigation

## Technologies Used

- HTML5
- CSS3
